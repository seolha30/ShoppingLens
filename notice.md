## 429에러 및 그록 안내
- 분당 할당량 또는 일일 할당량 만료로 뜨는 429에러.
  '어? 저는 이제 막 쓰기 시작했는데 떠요. 일일 할당량이 다되었다뇨?'
  아뇨!!! '분당 할당량' 문제가 제일 큽니다.
  분당은, 1분당이 아닌 N분당 입니다.
  그래서 1분/5분/15분/40분 알 수 없기에,
  과부하 에러인 503/타임아웃과 같다고 보시면 됩니다.
  해결책 : 다른 모델 사용 또는 재시도 (flow 또는 다른모델을 이용권장)
- 26.03.19 그록사이트 무료영상제작이 중단되었습니다.
  해외커뮤니티를 확인했을때, 아직은 두고봐야할것 같습니다.
  유료사용자도 광고팝업이 계속 뜨고 있어서, 그록사이트 버그인지, 정책변경인지 상황을 봐야할 듯하니 참고바랍니다.

## 롱폼관련 추가 안내
- 메이크렌즈는 Gemini api 기반이며, 로컬 프로그램입니다. 
롱폼은 쇼츠와 달리 대본이 긴 만큼, 많은 api 요청이 따르고, 인터넷 환경과 컴퓨터 사양이 너무 중요합니다.
대본이 길어 여러차례 gemini 요청중 1번이 끊기면 실패로 돌아가기도하고, 와이파이로 인해 인터넷환경이 좋지 않을시, 에러가 생길수도 있으며,
컴퓨터 사양이 낮은 CPU와 RAM 사용시, 시간이 엄청 오래걸리거나, 부하가 올 수 있습니다.
최대한 개선중이며, 위 내용을 꼭 인지하시고 스트레스가 덜 하시길 바라봅니다.
업그레이드는 계속 됩니드아~!!!
